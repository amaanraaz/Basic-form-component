# A basic login form component with ReactJs, Typescript and Tailwind CSS
### It includes field for email and password with a submit button

## Follow the below steps to run the project locally (assuming NodeJs and npm or yarn is preinstalled):

- Clone the repository:
  - `git clone <repo-url>`
- Navigate to project directory and install all dependencies:
  - `cd basic-login-project`
  - `npm install`
- Now in order to run the project just enter:
  - `npm run dev`
- Project will be up and running at localhost port:
  - http://localhost:5173/
